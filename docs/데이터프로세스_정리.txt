유동인구와 길거리 쓰레기
1.문제정의
- 유동인구가 늘어나면 길거리 쓰레기가 많아짐

2.가설
- 플로깅 모임을 촉진하는 커뮤니티를 만들면 길거리 쓰레기를 줄일수 있지 않을까??

-플로깅에 대한 뒷처리가 곤란한 경우 다수 발생
2. 가설
쓰레기통,산책로 데이터를 이어주면 뒷처리에 대한 문제가 해결되지 않을까?
-

현재 우리의 문제
-문제정의를 분석할 데이터가 없다.(가지고 있는데이터 : 뉴스기사,연관검색어)



해결방법
1. 문제를 정의할 수 있는 데이터 찾기 // 불가능하다고 봄
2. 새로운 문제정의 ( 새로운 데이터를 기반으로 분석을 한 후 새로운 문제정의)
3. 우리가 가지고 있는 데이터로 문제제기 하는것이 문제가 되지 않음.
(우리가 가진 데이터는 분석을 할 수 없음.)


데이터
유동인구가 적어지면 -> 쓰레기 줄어듬
유동인구 많아지면 -> 쓰레기 많아짐
